Ejemplos de Propiedades de Texto en CSS

1. Alineación del Texto

Este texto está alineado al centro. (text-align: center)

Este texto está alineado a la derecha. (text-align: right)

Este texto está justificado. Esto significa que el texto se ajusta para que las líneas tengan el mismo ancho, distribuyendo uniformemente los espacios. (text-align: justify)

2. Decoración del Texto

Este texto tiene una línea subrayada. (text-decoration: underline)

Este texto tiene una línea de tachado. (text-decoration: line-through)

3. Transformación del Texto

este texto está en mayúsculas. (text-transform: uppercase)

este texto está capitalizado. (text-transform: capitalize)

4. Espaciado del Texto

Este texto tiene un mayor espaciado entre letras. (letter-spacing: 5px)

Este texto tiene un mayor espaciado entre palabras. (word-spacing: 10px)

5. Altura de Línea

Este texto tiene una altura de línea de 2, lo que aumenta el espacio entre las líneas del párrafo. (line-height: 2)

6. Sombra de Texto

Este texto tiene una sombra gris aplicada. (text-shadow: 2px 2px 5px gray)
